# SQL Data Warehouse Project
This project showcases an end-to-end data warehousing solution built with MySQL Workbench. It covers loading raw data, cleaning and transforming it through a Medallion Architecture (Bronze, Silver, Gold), and writing SQL-based analytics to generate business insights.

### Data Architecture
This project follows the Medallion Architecture with three layers:

- Bronze: Raw data loaded directly from CSV files (ERP and CRM sources)
- Silver: Data is cleaned, standardized, and prepared for analysis
- Gold: Final data modeled into a star schema ready for reporting and analytics

### What This Project Covers
- Building a data warehouse from scratch
- Building ETL pipelines to extract, transform, and load data
- Data modeling using fact and dimension tables
- Writing SQL queries to analyze sales trends, customer behavior, and product performance
